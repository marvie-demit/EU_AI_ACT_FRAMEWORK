# Consent Withdrawal Procedure

1. **Request:** User submits a request to withdraw consent via [Link to form/email].
2. **Verification:** User identity is verified.
3. **Action:** User is removed from the biometric/emotion recognition system.
4. **Confirmation:** User receives confirmation of withdrawal within 72 hours.
5. **Data Deletion:** User data is deleted in accordance with GDPR.
