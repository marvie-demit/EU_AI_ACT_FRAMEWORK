# Registration Information Checklist (Based on Annex VIII)

**Source:** Activity_3_9


*(Use this checklist to gather all information in one place before starting the online registration.)*

| Section | Information Required | Source Document |
| :--- | :--- | :--- |
| **1. Provider Info** | Full legal name, address, and contact details of the provider. | Company Records |
| **2. System Info** | Trade name and any additional unambiguous reference allowing identification of the AI system. | Technical Docs |
| | Description of the intended purpose of the AI system. | Instructions for Use |
| | Status of the AI system (e.g., on the market, no longer on the market). | System Owner |
| **3. Conformity Info** | The unique registration number of the AI system. | (Provided upon registration) |
| | The EU Declaration of Conformity. | (Document from Activity 3.8) |
| | The Instructions for Use. | (Document from Activity 3.3) |
| | For third-party assessed systems, the name and number of the Notified Body and the certificate. | Certificate from Notified Body |
| **4. Digital Info** | Electronic copy of the Instructions for Use. | (Digital file) |
| | A URL for additional information (optional). | Marketing/Product |
