# GPAI Model Technical Documentation (Structure based on Annex XI)

**Source:** GPAI_Activity_1


*(This serves as a master table of contents for the full documentation package.)*

| Section | Title | Description | Owner |
| :--- | :--- | :--- | :--- |
| **1** | **General Description** | A detailed description of the GPAI model, its architecture, number of parameters, and modality (e.g., text, image). | Model Owner |
| **2** | **Training Process** | A detailed description of the training process, including the methodologies, tools, and computational resources used (including energy consumption). | ML Engineer |
| **3** | **Testing & Evaluation** | A detailed description of the testing and evaluation process, including the metrics used for performance, bias, and robustness, and the results of these tests. | Data Scientist |
| **4** | **Model Card** | A summary of the model's key characteristics, intended uses, limitations, and performance. (See template below). | Model Owner |
| **5** | **Instructions for Use** | The information to be provided to downstream providers to enable their compliance (covered in GPAI Activity 2). | Tech Writer |
