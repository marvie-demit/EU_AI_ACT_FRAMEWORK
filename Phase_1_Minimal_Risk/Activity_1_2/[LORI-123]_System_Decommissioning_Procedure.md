# AI System Decommissioning Procedure

1. **Request:** System Owner submits a decommissioning request.
2. **Approval:** AI Governance Body approves the request.
3. **Execution:** System is removed from production.
4. **Inventory Update:** System status is updated to "Decommissioned" in the inventory.
5. **Data Archival:** System data is archived or deleted per retention policy.
